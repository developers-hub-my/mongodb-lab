# Exercises

Three hands-on lab guides referenced by the slide deck. Each guide includes
the task list, mongosh commands, expected output, and trainer notes.

## Table of Contents

### [1. CRUD Practice](01-crud.md)

Lab Guide Part 2 — referenced from Slide 23. Load `books.json`, then create,
read, update, and delete documents. **Duration: 20 minutes.**

### [2. Querying and Indexing](02-querying-indexing.md)

Lab Guide Part 3 — referenced from Slide 26. Sort, project, and limit results;
create single-field, compound, and unique indexes; use `explain()` to confirm
index usage. **Duration: 25 minutes.**

### [3. Security Hardening](03-security-hardening.md)

Lab Guide Part 4 — referenced from Slide 42. Create least-privilege users,
verify access controls, take a `mongodump` backup, and restore from it.
**Duration: 30 minutes.**

## Prerequisite

All exercises assume you have completed [Getting Started](../01-getting-started/README.md)
and the lab stack is running.
